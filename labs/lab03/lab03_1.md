# Lab 03.1- Shortest Path

Try this GQL query:

```gql
-- Shortest Path query
MATCH p = ANY SHORTEST (src:`Person`)-[:`follows`]-{1,8}(dst:`Person`)
WHERE src.`name` = "Alice"
  AND dst.`name` = "Bob"
RETURN p, path_length(p) AS l
```

Then this one:

```gql
-- Shortest Path query
MATCH p = ANY SHORTEST (src:`Letters`)-[relatesTo:`relatesTo`]-{1,8}(dst:`Letters`)
WHERE src.`letters` = "a"
  AND dst.`letters` = "j"
RETURN
    TO_JSON_STRING(src) AS src,
    TO_JSON_STRING(dst) AS dst,
    TO_JSON_STRING(p) AS p,
    TO_JSON_STRING(path_length(p)) AS l
```


> [!TIP]
> `ALL SHORTEST` is not supported as at today.

# Lab 01.1- Shortest Path

Try this GQL query:

```gql
-- Shortest Path query
MATCH p = ANY SHORTEST (src:`Numbers`)-[relatesTo:`relatesTo`]-{1,8}(dst:`Numbers`)
WHERE src.`number` = 5
  AND dst.`number` = 6
RETURN p, path_length(p) AS l
```

> [!TIP]
> `ALL SHORTEST` is not supported as at today.
